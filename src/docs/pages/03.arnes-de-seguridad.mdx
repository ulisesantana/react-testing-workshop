# Creando un arnés de seguridad

Si queremos asegurarnos que a medida que añadimos cosas a este proyecto, éste no se rompe, 
tenemos que empezar sabiendo qué es lo que hace y crear tests que nos respalden.

El proyecto se compone de 3 vistas:
    - Assets
    - Holders
    - History

En la rama `start` tenemos la versión inicial del proyecto junto con los tests de Cypress para la vista de los Assets.
La idea de este taller es aprender los conceptos básicos para testear aplicaciones con React, por lo que por ahora con que testees la vista de Holders, basándote en la de Assets, servirá para entender los conceptos básicos.
