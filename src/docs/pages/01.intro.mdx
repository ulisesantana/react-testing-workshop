# Taller de testing con React

Hola!

Bienvenid@ a este taller sobre cómo testear aplicaciones hechas con React. Abordaremos las pruebas unitarias 
con <Link text={'react-testing-library'} to={'https://www.npmjs.com/package/@testing-library/react'}/> y 
las pruebas *end to end* con <Link text={'Cypress'} to={'https://www.cypress.io/'}/>.

Por lo pronto descarga el repositorio e instala todas las dependencias con:
```bash
git clone https://github.com/ulisesantana/react-testing-workshop.git
cd react-testing-workshop
yarn
```

Te recomiendo que sigas este taller en local con `yarn docz:dev`, ya que ahora mismo los enlaces relativos de docz no están funcionando como debería una vez haces el build.
